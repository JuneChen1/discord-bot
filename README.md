# Discord Bot

1. BOT_TOKEN  
[Applitions](https://discord.com/developers/applications) -> Bot -> TOKEN

2. CLIENT_ID  
[Applitions](https://discord.com/developers/applications) -> OAuth2 -> CLIENT ID

3. SERVER_ID  
Discord -> 使用者設定 -> 進階 -> 打開開發者模式  
伺服器 -> 右鍵點擊名稱 -> 複製ID

+ discord.js 14.2.0
+ @discordjs/rest 1.0.1
