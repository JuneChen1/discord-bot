# Discord Bot 指令說明

### `/remind`

設定提醒，將在事件前一天晚上 22:00（台灣時間）自動發送到指定頻道。

| 參數 | 必填 | 說明 |
|------|------|------|
| `date` | 是 | 事件日期，格式 `YYYYMMDD`，例如 `20260510` |
| `message` | 是 | 提醒內容 |
| `time` | 否 | 事件時間，格式 `HH:MM`，例如 `14:30` |

---

### `/reminders`

查看你所有待發送的提醒，包含事件日期、提醒時間、內容、頻道及 ID。

---

### `/remind-delete`

刪除一個待發送的提醒。

| 參數 | 必填 | 說明 |
|------|------|------|
| `id` | 是 | 提醒 ID（建立提醒時訊息底部可查到，或從 `/reminders` 查詢） |

---

### `/help`

列出所有可用指令與簡短說明。
